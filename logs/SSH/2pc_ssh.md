# 2PC SSH構築ログ（2026-04-21）

## 目的
2台のPCを用いてSSH接続を行う

---

## 構成
- クライアント：Mac（ターミナル使用）
- サーバ：Windows + VirtualBox（Ubuntu）
- ネットワーク：ブリッジ接続（ポートフォワーディング未使用）

---

## 手順

1. サーバ側のネットワーク設定でブリッジアダプターを指定  
2. サーバ側でSSHサービスとポートが開いているか確認  

3. クライアント側からSSH接続（初回はパスワード認証）  
   - サーバの公開鍵が `known_hosts` に未登録のため  
   - 接続時に自動で登録される  

4. 接続後、サーバ側で `.ssh` ディレクトリ確認  
   - パーミッションが適切であること（他ユーザに見えない）  

5. 一度接続を切断し、鍵認証の準備  
   - クライアント側で鍵ペア作成  

6. 公開鍵をサーバ側へ転送  
   - `authorized_keys` に追加  

7. 鍵認証で接続
```bash
ssh -i ~/.ssh/win_key user@IPアドレス
